# CSC-221, Adv. Python
Instructor - Hana Seidi

This course introduces advanced computer programming using the Python programming language. Emphasis is placed on advanced programming concepts, including advanced algorithms and programming principles, using standard and third-party libraries.

Upon completion, students should be able to design, code, test, and debug advanced Python programs. 

---

Grading of Programs (Python):
The purpose of this document is to help students understand assignment submission and grading
requirements. Below is a standardized list of how submitted programs are to be graded.

Program not named properly -10 points

No comment title block -10 points

Variable names not correct/meaningful (each variable) -5 points

Pseudocode missing or lacking (certain assignments) -10 points

Program doesn’t work as expected (logical errors) (see assignment rubrics)

Program has 3 or more of the above issues Grade of 25

Program doesn’t work at all (syntax errors) Grade of 1

Wrong program turned in Grade of 1

Second submission penalty -15

Using a while True loop at any time Grade of 1

Using any concept not yet covered in class Grade of 1
