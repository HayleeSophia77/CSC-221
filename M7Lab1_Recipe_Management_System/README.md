In this assignment you will demonstrate your understanding of using pyinstaller to convert programs to executable files.

1. First you are to Create a virtual environment and install pyinstaller in the environment.

2. Convert one of the programs you did for this class to an executable file.

Keep this simple. The following has to apply to the program you will convert:

* Requires user input.
* Displays results.
* Notifies user when it's going to stop/close. For instance:
	* if it's menu driven, display a message that says goodbye then close.
	* If it's a game, display a message that tells the user they lost then closes.
	* If none of the above, ask the user to hit enter or any key to close.
* Creating a Flask or Streamlit application is optional.

Important Requirements:

* Use functions.
* Unit testing must be done for at least 2 functions.
* Exception handling must be included.
